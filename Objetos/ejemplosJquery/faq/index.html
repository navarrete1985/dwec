FAQs

¿Qué es jQuery?

jQuery es una librería de funciones JavaScript que puedes utilizar para desarrollar sitios web.

¿Por qué es tan popular jQuery?

Hay tres razones:

  • Es gratuito
  • Te permite hacer más cosas en menos tiempo
  • Sus funciones son compatibles con todos los navegadores
